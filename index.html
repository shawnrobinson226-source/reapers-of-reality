The Pattern Loop

Event → Interpretation → State → Action → Reinforcement

Every repeated pattern follows a sequence. An event occurs. Meaning is assigned. A state forms. Action follows. The outcome reinforces the next interpretation. Once the loop is visible, it becomes interruptible.
